# Redes: dos conceitos iniciais à criação de uma intranet


- Ping

```
  ping www.google.com
  ping 8.8.8.8
  
```

- Tracerout

```
  // Windows
  tracert https://www.google.com/

  // Linux 
  tracerout https://www.google.com/
  
```

- DNS

```
  nslookup www.google.com
  
```

- ipconfig and ifconfig 

```

  // windows
  ipconfig /all

  // linux
  ifconfig
  ip addr show
  
```
